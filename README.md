# 🎮 Tic Tac Toe Game

A classic **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**. This project demonstrates JavaScript DOM manipulation, event handling, and game logic through an interactive two-player experience.

---

## 📖 Overview

The game allows two players to compete on a **3×3 grid**, alternating turns until one player wins or the game ends in a draw. It provides instant feedback, supports resetting the game, and offers a clean, responsive user interface.

---

## ✨ Features

- 🎲 Two-player gameplay (X vs O)
- 🏆 Automatic winner detection
- 🤝 Draw detection
- 🔄 Reset Game button
- 🆕 Start a New Game after completion
- 📱 Responsive layout for different screen sizes
- 🎨 Modern and clean user interface
- ⚡ Lightweight and fast (No external libraries)

---

## 🛠️ Built With

- **HTML5** – Structure
- **CSS3** – Styling & Responsive Design
- **JavaScript (ES6)** – Game Logic & DOM Manipulation

---

## 🎯 How to Play

1. Open the game in your browser.
2. Player **O** starts the game.
3. Players take turns placing their marks.
4. The first player to align **three symbols** horizontally, vertically, or diagonally wins.
5. If all nine cells are filled without a winner, the game ends in a draw.
6. Click **Reset Game** or **New Game** to play again.

---
### Run the project

Simply open **index.html** in your preferred web browser.

No installation or additional dependencies are required.
